# base64

```bash
Usage: strtools base64 [options] <command>

Options:
  -i, --input <string>  Input string
  -h, --help            display help for command

Commands:
  encode|enc: Encode string
  decode|dec: Decode string

  Examples:
    $ strtools base64 enc -i 'Hello World'
    $ echo -n 'Hello World' | strtools base64 encode
```
