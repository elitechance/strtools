# url

```bash
Usage: strtools url [options] <command>

Options:
  -i, --input <string>  Input string
  -h, --help            display help for command

Commands:
  encode|enc: Encode string
  decode|dec: Decode string

  Examples:
    $ strtools url enc -i 'Hello World'
    $ echo -n 'Hello World' | strtools url enc
```
